## Findings Requiring Follow-up

### FAIL — Style Quiz Down
`/style-quiz` shows: "The quiz is temporarily unavailable. Please try again shortly."  
This is a live prod failure, not a test artifact. Owner should investigate and restore.

### NOTE — Search Returns 0 Products
`/search?q=futon` returns 2 articles but 0 products. Product search index may not be populated or the Shopify search integration isn't connected. Needs investigation.

### NOTE — FAQ Entry Count Mismatch
TESTING-GUIDE says 17 FAQ entries; prod has 18. Guide needs updating or one entry should be reviewed for removal.

### NOTE — PDP Variant Picker Absent (Wilderness Log)
Wilderness Log Futon has no variant picker. This is expected if the product has only one variant or if all fabric variants are configured as separate product listings. Not a bug — confirms existing behavior matches Wix OOS suppression behavior pattern.

### NOTE — Velo Endpoints (Wix site)
Contact form and newsletter endpoints (`post_contactSubmissions`, `post_mailingListSignups`) require manual **Publish** in Wix Dashboard after every Velo code deploy. They were 404 during QA — needs manual publish step added to deploy runbook.
